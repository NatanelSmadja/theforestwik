# The Forest Companion V10

React + Vite companion app for The Forest.

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Vercel
- Framework: Vite
- Build Command: `npm run build`
- Output Directory: `dist`

## V10 Highlights
- Adventure Mode step-by-step game companion
- Smart Search across items, caves, enemies, weapons and crafts
- Craft Tree by material
- Inventory Simulator
- Build Planner
- Compare Lab for weapons
- Tips Hub and Survival Timeline
- Save Profiles
- Themes
- PWA / Offline files remain included
- Map Hub with custom map upload support

Note: real copyrighted game screenshots/assets are not bundled. Use your own approved/legal images if you want to add real visuals.
